# Win10装不上蓝牙驱动的问题记录


笔者的原因是因为系统中已经有一个蓝牙设备处于激活中了，把那个设备禁用了之后就好了。

似乎不禁用那个设备也可以安装驱动， 但是设备无法正常运转。 



两个蓝牙设备则分别是： 板载蓝牙和一个 PCI-e 的蓝牙设备。

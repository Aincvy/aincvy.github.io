# Java中Integer和int的相等判断


```java
int i == int j :  ✔️可以判断是否相等
Integer i == int j:    ✔️可以判断是否相等，因为会自动拆箱
Integer i == Integer j:   ❌不可以判断是否相等， 需要使用 equals			
```
